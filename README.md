# Car Price Prediction

## Overview

This project is a Machine Learning-based Car Price Prediction system that analyzes a dataset of car features and predicts the price of a car based on input parameters.

---

## Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Model training using machine learning algorithms
* Price prediction based on input features
* Jupyter Notebook for step-by-step workflow

---

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

---

## How to Run

### Option 1: Run Locally

1. Clone the repository:

```
git clone https://github.com/vrushalicbarhate/carprediction.git
```

2. Navigate to the folder:

```
cd carprediction
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the project:

```
python app.py
```

---

### Option 2: Run Notebook Online

1. Open Google Colab
2. Go to File → Open Notebook → GitHub
3. Paste the repository link
4. Open `models.ipynb`
